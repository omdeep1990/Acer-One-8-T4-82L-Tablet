## 🛠️ How to Flash

### Requirements
- Windows PC
- USB Cable (original recommended)
- SP Flash Tool (download above)
- Official Firmware (download above)
- MediaTek USB Driver (download above)

### Steps

1. **Install** MediaTek Driver — run `Mediatek_Driver_Auto_Installer_v1.1352.zip` as Administrator
2. **Download** and extract `Official.Firmware.rar` to a folder
3. **Open** `SP Flash Tool` (run as Administrator)
4. Click **"Choose"** and select `scatter.txt` from the extracted firmware folder
5. **Power off** your tablet completely
6. Click **"Download"** in SP Flash Tool
7. **Connect** tablet via USB (do NOT press power button)
8. Wait for the green checkmark — flash complete!
9. Tablet will **reboot automatically**
